# 欢迎来到我的博客

这是我的第一篇博客文章，欢迎阅读！

## 关于这个博客

这个博客使用 Next.js 构建，具有以下特性：

- **现代化设计**：使用 Tailwind CSS 构建响应式界面
- **Markdown 支持**：支持 Markdown 和 MDX 格式
- **深色模式**：支持深色/浅色主题切换
- **代码高亮**：内置代码语法高亮
- **标签系统**：文章可以添加多个标签

## 示例代码

```javascript
function hello() {
  console.log('Hello, World!');
}
```

## 为什么选择 Next.js？

Next.js 是一个强大的 React 框架，提供了：

1. 服务器端渲染（SSR）
2. 静态站点生成（SSG）
3. API 路由
4. 优秀的开发体验
